Metrics

Epic Online Services Interface: Metrics Interface

The Metrics Interface tracks your game's usage and populates the Game Analytics dashboard in the Developer Portal. This data includes active, online instances of the game's client and server, and past sessions played by local players.

Functions

These functions are provided for handling metrics:

  • eos_metrics_begin_player_session
  • eos_metrics_end_player_session

Constants

These are the constants used by this API:

  • EOS_METRICS_ACCOUNT_ID_TYPE
  • EOS_USER_CONTROLLER_TYPE

eos_metrics_begin_player_session

Epic Online Services Function: EOS_Metrics_BeginPlayerSession

This function logs the start of a new game session for a local player. The game client should call this function whenever it joins into a new multiplayer, peer-to-peer or single player game session. Each call to eos_metrics_begin_player_session must be matched with a corresponding call to eos_metrics_end_player_session.


Syntax:

eos_metrics_begin_player_session(account_id, display_name, account_id_type, controller_type, server_ip, game_session_id=undefined)
Argument Type Description
account_id String An Epic Account ID. Set this field when account_id_type is set to EOS_METRICS_ACCOUNT_ID_TYPE.EPIC.
display_name String The in-game display name for the user.
account_id_type EOS_METRICS_ACCOUNT_ID_TYPE Account ID type that is set in the union.
controller_type EOS_USER_CONTROLLER_TYPE The user's game controller type.
server_ip String IP address of the game server hosting the game session. For a localhost session, set to undefined. If both IPv4 and IPv6 addresses are available, use the IPv6 address.
game_session_id String Optional, application-defined custom match session identifier. If the identifier is not used, set to undefined which will be shown in the Played Sessions listing at the user profile dashboard.




Returns:

EpicResult


Example:

var _struct = eos_metrics_begin_player_session(
        account_id,
        "YYEpicGames",
        EOS_METRICS_ACCOUNT_ID_TYPE.EPIC,
        EOS_USER_CONTROLLER_TYPE.MOUSE_KEYBOARD,
        "No Server",
        "AnyRandomIDString");

if(_struct.status != EOS_RESULT.SUCCESS)
{
    show_debug_message("eos_metrics_begin_player_session failed");
}
The above code shows an example of how the function should be used.

eos_metrics_end_player_session

Epic Online Services Function: EOS_Metrics_EndPlayerSession

This function logs the end of a game session for a local player. Call once when the game client leaves the active game session. Each call to eos_metrics_begin_player_session must be matched with a corresponding call to eos_metrics_end_player_session.


Syntax:

eos_metrics_end_player_session(account_id, account_id_type)
Argument Type Description
account_id String Set this field when account_id_type is set to EOS_METRICS_ACCOUNT_ID_TYPE.EPIC
account_id_type EOS_METRICS_ACCOUNT_ID_TYPE The Account ID type that is set in the union




Returns:

EpicResult


Example:

var _struct = eos_metrics_end_player_session(account_id, EOS_METRICS_ACCOUNT_ID_TYPE.EPIC);
show_debug_message("eos_metrics_end_player_session: " + string(_struct.status_message));
The above code shows an example of how the function should be used.

EOS_METRICS_ACCOUNT_ID_TYPE

Epic Online Services Enum: EOS_EMetricsAccountIdType

These constants are used to describe the account ID type.

These constants are referenced by the following functions:

  • eos_metrics_begin_player_session
  • eos_metrics_end_player_session


Member Description
EPIC An Epic Account ID
EXTERNAL An external service Account ID

EOS_USER_CONTROLLER_TYPE

Epic Online Services Enum: EOS_EUserControllerType

These constants are used to describe the type of controller being used during the current player session.

These constants are referenced by the following functions:

  • eos_metrics_begin_player_session


Member Description
UNKNOWN The game controller type is unknown
MOUSE_KEYBOARD Mouse and keyboard controller
GAMEPAD_CONTROL Gamepad controller
TOUCH_CONTROL Touch controller
